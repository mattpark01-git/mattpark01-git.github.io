Git

Git Shortcuts
Push all changes to remote w/ generic commit msg
git add .; git commit -m "generic commit message"; git push;
Does all three commands in case I'm too lazy to put in a commit message.
Checking Updates
Checks if your local is behind remote
git fetch; git status --uno
Fetches from remote server, then tells you if local repo is behind.
Show log graph
git log --graph --all --pretty
Displays all logs in a nice graph format.
